# escolis-dashboard

## Lancement du projet : 
> npm start
    ou
> node server.js

Si erreur : 
- supprimer le dossier /node_modules/
> rm -R node_modules/
- reinstaller les modules : 
> npm i

(et normalement ça marche...)
